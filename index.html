There are 9 different types of D3 layouts. These include:

  1. Chord
  2. Cluster
  3. Force
  4. Pack
  5. Partition
  6. Pie
  7. Stack
  8. Tree
  9. Treemap

The 3 most common layouts include the pie, stack, and force layouts. To indicate which layout you want to use, you use d3.____. For example, to use a stack layout, you would use:

d3.stack()

The commonality between all of the layouts is that they all use the basic d3 algorithm. However, each layout performs a different function. For each layout you use the basic d3 algorithm and then add the certain elements that the layout needs to be visually effective.
